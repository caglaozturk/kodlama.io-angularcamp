## Kurulum

Rent a car projesinin çalışabilmesi için [Node.js](https://nodejs.org/) v13+ gereklidir.

Bağımlılıkları ve devDependencies'i kurun ve sunucuyu başlatın.

```sh
cd kodlama.io-angularcamp
npm i
ng serve --open
```
